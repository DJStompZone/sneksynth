## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/djstompzone/sneksynth.git
    ```

2. Navigate to the project directory:

    ```bash
    cd sneksynth
    ```

3. Install dependencies:

    ```bash
    pip install -r requirements.txt
    ```

4. Or install the package directly:

    ```bash
    pip install .
    ```
